Todo管理

シンプルで使いやすいTodo管理システムです。タスクの作成、編集、削除、完了状態の管理が簡単に行えます。

カテゴリー機能

Todoをカテゴリーで整理できます。プロジェクトや優先度別にタスクを分類して効率的に管理しましょう。

ダッシュボード

全体の進捗状況や統計情報を一目で確認できるダッシュボード機能付きです。

WAMBDA フレームワークについて

このアプリケーションはWAMBDAフレームワークを使用して開発されています。 WAMBDAは以下の特徴を持つサーバーレスWebアプリケーションフレームワークです：

  • AWS Lambda + API Gateway による完全サーバーレス構成
  • DynamoDB を使用したNoSQLデータベース
  • AWS Cognito による認証機能
  • Jinja2テンプレートエンジン
  • Django風のURL routing システム
