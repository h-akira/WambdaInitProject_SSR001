統合Todo管理

ダッシュボード、Todo管理、統計表示が一つに統合されたシンプルな管理システムです。 タスクの作成・編集・削除、完了状態の管理、フィルタリング機能、進捗統計をワンストップで利用できます。

統計表示 フィルタリング リアルタイム更新

カテゴリー管理

Todoをカテゴリーで整理・分類できます。プロジェクト別、優先度別、用途別などでタスクを効率的に管理し、 視覚的にもわかりやすいカラータグで識別できます。

プライマリー 成功 警告 重要

WAMBDA フレームワークについて

このアプリケーションはWAMBDAフレームワークを使用して開発されています。 WAMBDAは以下の特徴を持つサーバーレスWebアプリケーションフレームワークです：

  • AWS Lambda + API Gateway による完全サーバーレス構成
  • DynamoDB を使用したNoSQLデータベース
  • AWS Cognito による認証機能
  • Jinja2テンプレートエンジン
  • Django風のURL routing システム
